# Media Files

## Image Naming

Please move your images to this directory with the following names:

1. `image.png` → `gladiaster-agent-templates.png` (Predefined AI agent templates screenshot)
2. `image-1.png` → `gladiaster-agent-configuration.png` (Agent configuration settings screenshot)
3. `image-2.png` → `arena-agent-positions.png` (Agent positions and decision history screenshot)
4. `image-3.png` → `arena-agent-view.png` (Agent configuration view screenshot)

## Adding Videos

You can also add videos here. Supported formats:
- MP4, MOV (can be uploaded directly to GitHub)
- GIF animations
- YouTube links (use markdown thumbnail format in main README)
